# Hi there 👋, I'm Mateo Guamán Cueva

**Aerospace Engineering Student @ UPM | Computational Aerodynamics Intern**

I am a 4th-year Aerospace Engineering student at Universidad Politécnica de Madrid (UPM) and a Computational Aerodynamics Intern. I am passionate about Computational Fluid Dynamics (CFD), high-order methods, and scientific computing. My current work focuses on addressing the mathematical and computational bottlenecks of modern aerodynamic simulations, specifically in high-order mesh generation and volumetric deformation.

### 🚀 What I'm working on
* **BSc Thesis (TFG) & Research:** Developing an automated, high-fidelity mesh generation pipeline. I am working on translating linear unstructured meshes from commercial software into high-order polynomial meshes for spectral element solvers.
* **CFD Pre-processing:** Bridging the gap between geometric discretization and high-order solution accuracy using Python, `.cgns`, and HDF5 (`.h5`) formats.
* **Mesh Deformation:** Exploring and comparing mathematical models (such as Laplace and Neo-Hookean) to propagate surface deformations into internal volumetric meshes without element inversion.

* **Languages:** Python, C++ (Learning)

* **CFD & Data:** PyHOPE, HORSES3D, ParaView, HDF5, CGNS
* **Core Concepts:** High-Order Mesh Generation, Spectral Element Methods (SEM), Mesh Deformation Algorithms, Numerical Diffusion control.
